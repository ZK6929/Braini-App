## 📖 Overview

**Braini** is your all‑in‑one AI‑powered study companion – built as part of the **Build with AI** workshop by **GeeksforGeeks** in collaboration with **Google**. Upload any PDF or TXT document and instantly unlock a suite of intelligent learning tools: AI chat, mind maps, flashcards, quizzes, podcasts, presentations, rich notes, annotations, read‑aloud, and gamified progress tracking – all powered by **Google Gemini 2.5 Flash**.

Upload PDFs or text documents and instantly generate:

- 📚 Smart summaries
- 🧠 Mind maps
- 🎴 Flashcards
- ❓ Interactive quizzes
- 🎧 AI podcasts
- 🎤 Presentations
- 📝 Notes
- 🔍 AI-powered document chat

Everything runs inside a beautiful responsive interface with dark/light mode, accessibility settings, and productivity-focused tools.

## 🚀 Getting Started (Local Development)

### 1. Clone the repository

```bash
git clone https://github.com/ZK6929/Braini-App.git
cd Braini-App
```

### 2. Open the application

Simply open **`index.html`** in your favorite web browser.

> ✅ No build tools  
> ✅ No package manager  
> ✅ No installation required  
> ✅ Runs entirely in the browser

---

## 🔑 Get Your Free Gemini API Key

1. Visit **Google AI Studio**
2. Sign in with your Google account.
3. Click **"Get API Key"**.
4. Create a new API key (it's completely free and requires **no credit card**).
5. Copy your API key.

---

## 🔗 Connect Braini

1. Open the Braini app.
2. Paste your Gemini API key into the **API Key** input field in the **top-right corner**.
3. When connected successfully, the **status indicator turns green**.
